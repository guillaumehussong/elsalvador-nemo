Focus group virtuel

Réactions B2C · personas Nemotron · El Salvador

What is this?

This tool runs a virtual focus group against synthetic Salvadoran consumer personas from the Nemotron dataset. You paste a stimulus (ad, offer, hook); the system samples N matching personas and asks an LLM to role-play each one in Salvadoran Spanish.

  • You get: mean interest score (1–10), sentiment breakdown, recurring objections, and sample verbatim quotes.
  • Use it for: early messaging tests, price sensitivity, objection mining — before spending on real field research.
  • Not a substitute for: real users, legal/compliance review, or statistically representative surveys.

Each persona = one LLM call (~8 s). Estimate cost before running. Responses are in Spanish; write your stimulus in Spanish when possible.

Configure a run

Estimate cost — token/cost preview, no LLM calls. Run focus group — executes N parallel persona interviews and saves JSON under data/runs/ on the server.

N > 15 : cochez pour confirmer le coût avant de lancer. Runs above 15 personas require explicit confirmation because LLM cost and latency scale linearly.
